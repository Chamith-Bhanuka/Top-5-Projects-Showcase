# Top-5-Projects-Showcase

### 1. ROSCA Financial Management System
**Tech Stack:** MongoDB, Express.js, React.js, Node.js (MERN), Stripe API
* **Description:** A full-stack FinTech platform designed to digitize Rotating Savings and Credit Association operations with secure payment integration.
* **Key Features:** JWT authentication, RBAC, and an AI-powered conversational agent for group matching.
* **Links:** [Frontend Repo](https://github.com/Chamith-Bhanuka/rosca-savings-system-fe) | [Backend Repo](https://github.com/Chamith-Bhanuka/rosca-savings-system-be) | [Live Demo](https://rosca-savings-system-fe.vercel.app/)

### 2. Smart Plant Management System
**Tech Stack:** Java Spring Boot, Python, ESP32, MySQL, Ollama AI
* **Description:** A distributed IoT system for real-time environmental monitoring and AI-driven plant disease diagnosis.
* **Key Features:** Real-time sensor data ingestion and automated health analysis using Python-based AI services.
* **Links:** [GitHub Repository](https://github.com/Chamith-Bhanuka/Smart-Plant-Management-System)

### 3. ChromaStyle AI Wardrobe Planner
**Tech Stack:** React Native (Expo), Firebase, Zustand
* **Description:** A cross-platform mobile application for wardrobe digitization featuring real-time native color extraction.
* **Key Features:** Native image processing to extract dominant hex code palettes from physical fabrics.
* **Links:** [GitHub Repository](https://github.com/Chamith-Bhanuka/ChromaStyle) | [Download APK/Expo Build](https://expo.dev/accounts/chamith_bhanuka/projects/ChromaStyle/builds/e663fa30-bd57-4974-96ae-5f282320f143)

### 4. Industrial Energy Consumption Monitor
**Tech Stack:** Java, JavaFX, MySQL, Java Serial Communication
* **Description:** A desktop application for real-time visualization and management of industrial machinery energy usage.
* **Key Features:** Implements UART-style serial protocols to bridge software UI with simulated hardware machinery.
* **Links:** [GitHub Repository](https://github.com/Chamith-Bhanuka/energy-consumption-monitor)

### 5. FX Chat Application
**Tech Stack:** Java, JavaFX, Sockets (TCP/IP), MySQL
* **Description:** A multi-client desktop chat application utilizing socket programming for real-time communication.
* **Key Features:** Secure message transmission and a modular UI designed with JavaFX.
* **Links:** [GitHub Repository](https://github.com/Chamith-Bhanuka/Fx-Chat-Application)
